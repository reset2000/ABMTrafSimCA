◆ checkCorrectProbabilitesAndCounts()

QString MainTrafficSimulator::checkCorrectProbabilitesAndCounts ( )
private

A method that checks the probabilities and counts for car lengths and driver profiles.

This method checks if all the probabilities for car lengths and driver profiles sum up to 100%. Additionally it also verifies if the number of different chosen lengths or driver profiles doesn't exceed maximum value of 10 (counted separately).

Returns
String with warning message (null if operation was compleated successfully)

◆ on_cbApproximate_stateChanged

void MainTrafficSimulator::on_cbApproximate_stateChanged ( )
privateslot

Method that activates after changing the state of the "Approximate" checkbox on the "Fundamental diagram" tab.

This method is responsible for changing the state of the trend lines shown on "Fundamental diagram" tab. It alternates between 3 possible states: no line, line connecting all the density points or approximated line (with the help of Eigen library)
