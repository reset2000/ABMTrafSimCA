\doxysubsection{Detailed Description}
The \doxylink{class_road}{Road} class contains all the functions neccessary to carry out the simulation. 

The \doxylink{class_road}{Road} class is the main component of the simulation engine. It contains all the functions that are called during this process. It also includes all the necessary components, such as road created out of semgents and cars that are driving through the lanes.
